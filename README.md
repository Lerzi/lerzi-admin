# lerzi-admin

## 项目安装

```sh
pnpm install
```

### 开发环境

```sh
pnpm dev
```

### 生产环境

```sh
pnpm build
```

### 单元测试[Vitest](https://vitest.dev/)

```sh
pnpm test:unit
```

###  [ESLint](https://eslint.org/)

```sh
pnpm lint
```
